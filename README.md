## 🤖 AI Multi-Site Web Scraper

An intelligent web scraping tool built with Python and Streamlit that lets you extract 
structured data from multiple websites simultaneously using plain English queries.

No coding. No CSS selectors. Just describe what you want — AI does the rest.

### ✨ Features
- 🌐 Scrape multiple websites at once
- 🧠 Powered by Groq's FREE LLaMA 3.3 70B model
- 💬 Natural language queries — type in plain English
- 📊 Export results as Excel or JSON
- ⚡ Real-time live logs
- 🎨 Dark hacker-style UI

### 🛠️ Tech Stack
- Python + Streamlit
- Groq LLaMA 3.3 70B (Free API)
- httpx + BeautifulSoup
- pandas + openpyxl

### 🚀 Quick Start
1. Get free API key at console.groq.com
2. pip install -r requirements.txt
3. streamlit run app.py
4. Enter API key + URL + query → Download data!
